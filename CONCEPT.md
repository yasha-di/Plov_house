# CONCEPT — Плов Хаус (адрас/атлас)

## Проект
Лендинг для заказа **узбекского плова** с доставкой. Одностраничник: Hero → виды
плова (4 карточки) → как заказать → традиция → футер. Форма заказа в модалке
(имя, телефон, выбор плова), пока **без реальной отправки**.

## Визуальная идея — АДРАС / АТЛАС (ikat)
Дизайн вдохновлён узбекскими шёлковыми тканями **адрас** и **атлас** (техника
_ikat / абрбанд_):

- **Вертикальные перетекающие полосы** с размытыми, «растёкшимися» границами —
  фирменная нечёткость ikat, где краска расплывается по основе.
- **Насыщенные, праздничные цвета:** малиновый, изумрудный, бирюзовый, жёлтый,
  фиолетовый, оранжевый. Это **не** золото-минимализм — это смело, нарядно,
  традиционно, но подано современно.
- На **тёмной основе** (тёплый уголь), чтобы цвета светились и не утомляли.

## Фишка — «живая ткань»
Цвета адрас/атлас **медленно переливаются и меняют оттенки** (анимированный
градиент + плавный сдвиг hue по бесконечному циклу). Ощущение, будто ткань
дышит. Фон **реагирует на скролл** (параллакс + смещение палитры).

## Анимации (motion)
- Фон: бесконечный цикл смены оттенков ikat-полос + параллакс/сдвиг по скроллу.
- **Рисопад**: сверху падают реалистичные ингредиенты — рис, морковь соломкой,
  кусочки мяса, зира, изюм, нут. У каждого своя форма, цвет, скорость, вращение
  и покачивание. ~30–40 частиц ради плавности.
- Hero: появление заголовка/слогана, переливающийся ikat-акцент, кнопка «Заказать».
- Карточки: hover со сдвигом цвета и подсветкой ikat.
- Формы/кнопки: микровзаимодействия 200–300 мс.

## Технологии
React + Vite, `motion/react` (framer-motion API), инлайн-стили + CSS-переменные
в `src/index.css`. **Без Tailwind**. Шрифты: Playfair Display (заголовки) + Inter (текст).

## Правила
- Уважать `prefers-reduced-motion`: отключать бесконечные циклы и падение частиц.
- Анимировать только `transform` / `opacity` / `filter` ради производительности.
- Контраст текста ≥ 4.5:1, видимые focus-состояния, `aria` для иконочных кнопок.
- Адаптив: 375 / 768 / 1024 / 1440.
