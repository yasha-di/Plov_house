The my-site plov landing pivoted away from the original "dark gold minimalism + suzani ornament" look to an **adras/atlas (Uzbek ikat) fabric** theme: bold vibrant vertical flowing ikat stripes with blurred "bleeding" edges; saturated colors (crimson, emerald, turquoise, yellow, purple, orange) over a dark base. Signature effect: colors must **slowly cycle/shift hue** (animated gradient) so the fabric feels alive. The background ikat reacts to scroll. Falling-ingredient particles must look realistic (rice, julienned carrot, meat, cumin, raisin, chickpea), ~30-40 max for smoothness. Built all in code with motion + the ui-ux-pro-max skill (style: "Vibrant & Block-based"). Always respect prefers-reduced-motion and watch performance. See [[user-prefers-no-magic-mcp]].
